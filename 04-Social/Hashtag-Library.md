# Hashtag Library

Use this page to organize hashtags by purpose. Hashtags should support discoverability and context without replacing clear captions.

## Categories

Maintain groups for:

- Chapter identity
- Motorcycle community
- Event type
- Location
- Partner or sponsor campaigns
- Seasonal or recurring events

## Review Standard

Before using a hashtag:

- Confirm it is still active and appropriate.
- Confirm it does not attach the chapter to unrelated or harmful content.
- Avoid using partner or campaign hashtags without context.
- Keep event-specific hashtags documented with the event archive.

## Usage

Use hashtags intentionally. A short, relevant set is easier to maintain than a long list that no one checks.
