# Logo Guidelines

This page covers operational handling for official and chapter logo files. It does not define global logo rules.

## Storage

Store reusable logo files in [../03-Media/Assets/Logos](../03-Media/Assets/Logos/). Keep source files, transparent exports, and dated versions organized according to [../03-Media/Logo-Library.md](../03-Media/Logo-Library.md).

## Handling

- Use the highest-quality approved file available.
- Keep source files separate from one-time exports.
- Do not overwrite materially different versions.
- Do not redraw official marks.
- Do not stretch or distort logo files.
- Confirm partner or sponsor usage before publishing.

## File Types

Preferred formats:

- SVG for scalable digital use
- PNG for transparent-background exports
- AI or PDF when provided as editable or vector source
- JPG only when no better format exists

## Approval

When logo usage involves merch, sponsors, paid promotions, or global representation, confirm approval before production.
