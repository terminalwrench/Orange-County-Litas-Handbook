# Quarterly Planning

Quarterly planning helps leadership turn annual goals into manageable work.

Use [Templates/Quarterly-Review.md](Templates/Quarterly-Review.md) at the end of each quarter.

## Quarterly Inputs

Review:

- Annual goals
- Upcoming events
- Event recaps and lessons learned
- Member feedback themes
- Open action items
- Partnership opportunities
- Route and ride lead needs
- Documentation updates

## Quarterly Plan

For each quarter, define:

- Priority focus
- Key events
- Leadership assignments
- Risks or capacity constraints
- Communication needs
- Handbook updates needed
- Review date
