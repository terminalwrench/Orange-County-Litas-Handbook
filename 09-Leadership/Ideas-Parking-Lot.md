# Ideas Parking Lot

Use this file for leadership ideas that may become future work but are not active commitments.

## Idea Format

```markdown
## Idea Title

- Source:
- Purpose:
- Possible owner:
- Related module:
- Timing:
- Decision needed:
```

## Review

Review the parking lot during quarterly planning. Move ideas into active planning only when there is:

- A clear purpose
- A realistic owner
- Enough capacity
- A production or event path

Archive ideas that are no longer relevant.
