# Reel Checklist

Use this checklist before publishing an OC Litas reel.

## Content Readiness

- [ ] Final video export is approved.
- [ ] Reel cover is approved.
- [ ] Caption is final.
- [ ] Event details are accurate.
- [ ] Required tags are confirmed.
- [ ] Collaborator accounts are confirmed.
- [ ] Audio is available and appropriate for the platform.
- [ ] Any spoken information is captioned or summarized.

## Technical Check

- [ ] Video is vertical when intended for reels.
- [ ] Export file opens correctly.
- [ ] Cover image matches [../Image-Sizes.md](../Image-Sizes.md).
- [ ] Cover crop has been checked for feed and profile views.
- [ ] Final file uses [../Naming-Conventions.md](../Naming-Conventions.md).
- [ ] Draft exports are separate from the final export.

## Accessibility

- [ ] Captions are included when speech carries important information.
- [ ] Caption text includes key event or recap details.
- [ ] Important visual information is not only conveyed through audio.
- [ ] Flashing or visually intense clips are avoided or reviewed carefully.

## Publishing

- [ ] Chapter account is correct.
- [ ] Venue, photographer, partner, or sponsor tags are correct.
- [ ] Collaborator invite is ready if used.
- [ ] Posting time is confirmed.
- [ ] Cover image is selected before publishing.
- [ ] Live reel is checked after publishing.
- [ ] Collaborator invite acceptance is checked.

## Archive

- [ ] Final video export is saved.
- [ ] Final cover image is saved.
- [ ] Caption is saved.
- [ ] Posting date is recorded.
- [ ] Audio or music note is recorded when useful.
- [ ] Tagged and collaborator accounts are recorded.
- [ ] Final files are linked from the event recap or media archive.
