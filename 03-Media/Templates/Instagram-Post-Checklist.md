# Instagram Post Checklist

Use this checklist before publishing an OC Litas Instagram feed post or story.

## Event Details

- [ ] Event name is correct.
- [ ] Date and day of week are correct.
- [ ] Time is correct.
- [ ] Meetup and kickstands-up times are listed separately for rides.
- [ ] Location or destination is correct.
- [ ] Address, RSVP, or registration instructions are included when needed.
- [ ] Cost, donation, or ticket details are included when needed.
- [ ] Ride status is clear.
- [ ] Beginner-friendly status is clear when relevant.
- [ ] Parking note is included when needed.
- [ ] Weather or cancellation timing is included when relevant.

## Asset Check

- [ ] Export size matches [../Image-Sizes.md](../Image-Sizes.md).
- [ ] Final file uses [../Naming-Conventions.md](../Naming-Conventions.md).
- [ ] Required logos are included.
- [ ] Required partner or sponsor assets are approved for use.
- [ ] Important information is not hidden by platform interface areas.
- [ ] Final file is stored in the correct working or archive location.

## Caption And Accessibility

- [ ] Caption includes essential event details.
- [ ] Image-only text is repeated in the caption.
- [ ] Alt text is added when practical.
- [ ] Abbreviations are clear to new attendees.
- [ ] Any safety, ride, or RSVP requirement is easy to find.

## Tags And Publishing

- [ ] Chapter account is correct.
- [ ] Venue tag is correct if used.
- [ ] Partner tags are correct if used.
- [ ] Sponsor tags are correct if used.
- [ ] Photographer credit is included when required.
- [ ] Collaborator invite is prepared when needed.
- [ ] Post time or schedule is confirmed.
- [ ] Live post is checked after publishing.

## Archive

- [ ] Final asset is saved.
- [ ] Final caption is saved.
- [ ] Posting date is recorded.
- [ ] Tagged accounts are recorded.
- [ ] Asset is linked from the event recap or media archive.
