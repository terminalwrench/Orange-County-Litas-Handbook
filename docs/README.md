# GitHub Pages Site

This folder contains a lightweight static landing page for the Orange County Litas Handbook.

It uses plain HTML and CSS only:

- `index.html`
- `styles.css`

No React, build tools, package manager, or app framework is required.

## Purpose

The page gives cofounders and chapter leaders a friendly way to browse the handbook without needing to understand the GitHub repository structure first.

## GitHub Pages

To publish this page with GitHub Pages, configure Pages to serve from the `docs/` folder on the repository branch you use for publishing.

The page links back to the existing Markdown files using relative paths. It does not rename, move, or replace handbook content.
