# Shot List

Use this shot list as an operational reminder for event coverage. It does not dictate artistic style.

## Event Coverage

- Venue or destination context
- Bikes arriving
- Group parking
- First-timer welcome moments
- Ride lead or host
- Candid group interaction
- Group photo
- Partner or sponsor details
- Safe bike details
- Recap-ready closing images

## Photo Shoots

- Check-in
- Staging
- Individual rider portraits
- Bike portraits
- Detail shots
- Delivery reference notes
