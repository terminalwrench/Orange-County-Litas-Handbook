# Orange County Litas Portal

Welcome to the Orange County Litas Portal: a friendly website version of the chapter handbook.

This portal keeps chapter knowledge organized so current and future leadership can find what they need without digging through raw files. It is built from the existing Markdown handbook, which remains the source of truth.

You do not need to know GitHub to use this portal. Click a section, find what you need, and send updates to chapter leadership.
