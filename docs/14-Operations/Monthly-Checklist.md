# Monthly Checklist

Use this checklist at the beginning or end of each month to keep chapter operations current.

## Calendar

- [ ] Review upcoming 30 to 60 days.
- [ ] Confirm event owners.
- [ ] Check for community event conflicts.
- [ ] Confirm ride, meetup, collaboration, and special event balance.
- [ ] Identify events that need venue, route, partner, or asset follow-up.

## Events

- [ ] Update completed event records in [../10-Events-Database](../10-Events-Database/README.md).
- [ ] Confirm upcoming event details before public posting.
- [ ] Capture event lessons.
- [ ] Archive final event materials.
- [ ] Add follow-up tasks to meeting notes or the yearly planning board.

## Routes and Venues

- [ ] Update venue notes after events.
- [ ] Update route notes after completed rides.
- [ ] Review parking, fuel, restroom, and weather issues from recent events.

## Partnerships

- [ ] Review active partner or sponsor commitments.
- [ ] Send thank-yous or follow-ups.
- [ ] Update [../11-Venues](../11-Venues/README.md) records when needed.

## Media and Assets

- [ ] Confirm final flyers, photos, captions, and recap assets are stored or linked.
- [ ] Move reusable assets to the correct media location.
- [ ] Remove duplicate drafts from working areas when appropriate.

## Leadership

- [ ] Review open action items.
- [ ] Review decisions that need documentation.
- [ ] Check leader workload and upcoming handoffs.
- [ ] Identify any volunteer support needed.

## Knowledge

- [ ] Add reusable lessons to [../13-Knowledge/Lessons-Learned.md](../13-Knowledge/Lessons-Learned.md).
- [ ] Update FAQs, common challenges, or resources if new patterns appeared.
