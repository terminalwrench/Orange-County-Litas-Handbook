# Lessons Learned

Use this log to capture operational improvements after OC Litas events. The goal is not to criticize people. The goal is to make future events easier, safer, warmer, and more consistent.

Update this file after every event recap. Link back to the event record when one exists.

## Entry Format

Use this structure for each event:

```markdown
## YYYY-MM-DD - Event Name

- Event type:
- Location:
- Attendance estimate:
- Owner:
- What worked:
- What should change:
- Venue notes:
- Ride notes:
- Weather notes:
- Communication notes:
- Photo or asset notes:
- Follow-up actions:
```

## Operating Themes To Watch

Track patterns across events:

- Venues that consistently work well
- Venues that become difficult at certain group sizes
- Routes that are better for experienced riders than beginners
- Parking issues
- Weather timing problems
- Confusing public copy
- Flyer or announcement timing
- Volunteer role clarity
- Photo delivery timing
- First-timer experience

When the same lesson appears repeatedly, update the relevant operating document instead of leaving the improvement only in this log.
