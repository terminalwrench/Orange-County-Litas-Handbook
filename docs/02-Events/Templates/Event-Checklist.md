# Event Checklist

Use this checklist for OC Litas event planning. Not every item applies to every event, but every skipped item should be intentionally skipped.

## Idea

- [ ] Event purpose is clear.
- [ ] Event type is identified.
- [ ] Event owner is assigned.
- [ ] Expected audience is defined.
- [ ] Date or date range is realistic.
- [ ] Ride status is known.

## Planning

- [ ] Event brief is created.
- [ ] Day-of host is assigned.
- [ ] Budget, cost, or payment needs are known.
- [ ] Volunteer roles are identified.
- [ ] Photography needs are identified.
- [ ] Weather risks are reviewed.
- [ ] Cancellation owner is assigned.

## Venue

- [ ] Venue or destination is selected.
- [ ] Address is confirmed.
- [ ] Parking is confirmed.
- [ ] Reservation or group approval is confirmed.
- [ ] Food, drink, restroom, and access details are confirmed.
- [ ] Venue rules are documented.
- [ ] Backup plan is identified if needed.

## Ride

- [ ] Ride Lead is assigned.
- [ ] Sweep Rider is assigned.
- [ ] Route is validated.
- [ ] Meetup location is confirmed.
- [ ] Meetup and kickstands-up times are separate.
- [ ] Gas and restroom plan is confirmed.
- [ ] Regroup points are identified.
- [ ] Beginner suitability is confirmed.
- [ ] Emergency plan is reviewed.
- [ ] Weather decision time is set.

## Communications

- [ ] Flyer request includes final details.
- [ ] Public caption is drafted.
- [ ] Email is drafted if needed.
- [ ] RSVP or payment instructions are clear.
- [ ] Parking instructions are included.
- [ ] Beginner or first-timer notes are included.
- [ ] Weather or cancellation note is included.
- [ ] Original post is updated if details change.

## Day Of

- [ ] Host arrives early.
- [ ] Venue conditions are checked.
- [ ] Parking flow is checked.
- [ ] Volunteers know their roles.
- [ ] First-timers are welcomed.
- [ ] Ride briefing is completed if applicable.
- [ ] Photos are captured.
- [ ] Venue is thanked.
- [ ] Issues are noted for recap.

## Recap

- [ ] Attendees are thanked.
- [ ] Venue or partners are thanked.
- [ ] Social recap is posted.
- [ ] Photos or asset links are saved.
- [ ] Attendance estimate is recorded.
- [ ] Venue notes are recorded.
- [ ] Route notes are recorded if applicable.
- [ ] Lessons learned are added.
- [ ] Final materials are archived.
