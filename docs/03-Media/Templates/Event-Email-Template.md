# Event Email Template

Use this as a general starting point for event communications.

## Subject

```text
[OC Litas] [Event Name] - [Date]
```

## Email Draft

```text
Hi everyone,

We are sharing details for [event name].

Date:
Time:
Location:
Address:
Hosted by:
Partner(s):

What to know:
- [Requirement or note]
- [Requirement or note]
- [Requirement or note]

Please reply or message leadership if you have questions.

See you soon,
Orange County Litas
```

## Send Checks

- [ ] Event details match the related event record.
- [ ] Location and address are verified.
- [ ] Rider requirements are clear.
- [ ] RSVP or registration instructions are included when needed.
- [ ] Partner names are spelled correctly.
- [ ] Final proofread completed.
