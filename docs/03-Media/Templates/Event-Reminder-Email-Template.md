# Event Reminder Email Template

Use this for short reminders before events, rides, and meetups.

## Subject

```text
Reminder: [Event Name] is [date or relative timing]
```

## Email Draft

```text
Hi everyone,

Quick reminder that [event name] is coming up.

Date:
Time:
Location:
Address:

Before you arrive:
- [Reminder]
- [Reminder]
- [Reminder]

Check the original event details for anything that changes before the event.

Orange County Litas
```

## Send Checks

- [ ] Original event details verified.
- [ ] Weather or schedule changes included.
- [ ] Parking, RSVP, or route reminders included when needed.
- [ ] Final proofread completed.
