# General Social Media Checklist

Use this before publishing any chapter post, story, reel, or event graphic.

## Event Details

- [ ] Date is correct.
- [ ] Time is correct.
- [ ] Location is correct.
- [ ] Address or meetup point is correct.
- [ ] RSVP, registration, or ticket instructions are clear.
- [ ] Rider requirements are included when applicable.

## Credits And Tags

- [ ] Photographer credit confirmed.
- [ ] Designer credit confirmed when needed.
- [ ] Partner tags verified.
- [ ] Venue tags verified.
- [ ] Instagram tags verified.
- [ ] Hashtags selected from [../../04-Social/Hashtag-Library.md](../../04-Social/Hashtag-Library.md).

## Accessibility And Proofing

- [ ] Text is readable on mobile.
- [ ] Important text is not blocked by platform controls.
- [ ] Alt text, captions, or accessible description are included when practical.
- [ ] Names are spelled correctly.
- [ ] Final proofread completed.

## Archive

- [ ] Final file is stored in the correct location.
- [ ] Related event record is updated when applicable.
