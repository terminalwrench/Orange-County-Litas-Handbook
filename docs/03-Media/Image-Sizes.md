# Image Sizes

Consistent export sizes help media appear clean across platforms, reduce last-minute resizing, and make archived deliverables easier to reuse. These sizes are technical recommendations only. They do not define the creative style of any asset.

Use this with [Instagram-Guidelines.md](Instagram-Guidelines.md), [Video-Standards.md](Video-Standards.md), and [Templates/Flyer-Brief.md](Templates/Flyer-Brief.md).

## Instagram Posts

Recommended exports:

| Format | Size | Use |
| --- | --- | --- |
| Square post | `1080 x 1080 px` | Standard feed graphics and simple announcements. |
| Portrait post | `1080 x 1350 px` | Feed graphics that need more vertical space. |
| Landscape post | `1080 x 566 px` | Wide photo or banner-style feed assets. |

Portrait posts often use feed space efficiently, but the chosen format should match the deliverable request and available media.

## Instagram Stories

Recommended export:

| Format | Size | Use |
| --- | --- | --- |
| Story | `1080 x 1920 px` | Story announcements, reminders, and event updates. |

Keep important event information away from the top and bottom interface zones where platform controls may cover content.

## Reels Cover Images

Recommended exports:

| Format | Size | Use |
| --- | --- | --- |
| Reel cover | `1080 x 1920 px` | Full vertical cover for the reel view. |
| Feed-safe preview area | `1080 x 1350 px` centered within the cover | Helps key information remain visible in feed preview. |
| Profile grid safe area | Center square within the cover | Helps the cover remain recognizable in the profile grid. |

Designers should confirm how the cover crops in the platform before publishing.

## Facebook Event Images

Optional recommended export:

| Format | Size | Use |
| --- | --- | --- |
| Facebook event cover | `1920 x 1005 px` | Event page cover image. |

If Facebook is not part of the event promotion plan, this export is not required.

## Print Flyers

Optional recommended setup:

| Format | Size | Use |
| --- | --- | --- |
| Letter flyer | `8.5 x 11 in` | Printable flyer. |
| Half-page flyer | `5.5 x 8.5 in` | Smaller handout. |

For print, confirm bleed, trim, color profile, and resolution with the printer or vendor before final export. A common print target is `300 DPI`, but vendor requirements should control the final file.

## Why Consistency Matters

Consistent sizing:

- Reduces repeated export work
- Makes files easier to identify
- Helps event announcements look technically clean
- Prevents platform cropping from hiding event details
- Supports reliable archiving
- Allows future leaders to reuse files without guessing

When a platform changes requirements, update this document and note the date of the change in the related event or media workflow.
