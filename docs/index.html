Orange County Litas

A simple portal for chapter knowledge.

This portal keeps the chapter's operating notes, event records, routes, venues, leadership resources, and history in one place so future leaders can find what they need without digging through old chats or guessing how things were done.

You do not need to know GitHub to use this portal. Click a section, find what you need, and send updates to chapter leadership.

For Cofounders

You do not need to know GitHub to use this.

Click a section, read what you need, and send updates to leadership when something should be corrected or added.

This is meant to make chapter work easier, not more complicated. Think of it like a shared binder: routes, events, venues, history, and operating notes are organized so the next person does not have to start from scratch.
