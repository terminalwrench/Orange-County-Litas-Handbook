# New Venue Evaluation

Use this checklist before choosing a new venue for an Orange County Litas event. A venue should support the purpose of the event, the rider experience, and the chapter's ability to host safely and respectfully.

## Basic Fit

- [ ] Venue name is confirmed.
- [ ] Address is confirmed.
- [ ] Hours match the proposed event time.
- [ ] Venue is open to the public or chapter access is approved.
- [ ] Venue category is clear.
- [ ] Event purpose matches the venue environment.
- [ ] Venue has enough space for the expected group.

## Parking and Motorcycles

- [ ] Motorcycle parking is available.
- [ ] Parking surface is suitable for motorcycles.
- [ ] Entry and exit are safe for group arrivals.
- [ ] Parking does not require difficult U-turns or steep low-speed maneuvers.
- [ ] Parking cost, validation, permit, or time limit is understood.
- [ ] Overflow parking is identified if needed.

## Rider Experience

- [ ] Venue is appropriate for newer riders if the event is beginner-friendly.
- [ ] Route to the venue matches the advertised difficulty.
- [ ] Venue can handle staggered arrivals.
- [ ] Riders can safely gear down, wait, and regroup.
- [ ] Restrooms are available.
- [ ] Water, food, shade, or indoor space is available when relevant.

## Operations

- [ ] Reservation requirements are confirmed.
- [ ] Group size expectations are communicated to the venue.
- [ ] Payment, tab, ordering, or seating process is understood.
- [ ] Outdoor seating availability is known.
- [ ] Weather backup is considered.
- [ ] ADA access notes are reviewed.
- [ ] Contact name and phone or email are recorded if available.

## Community Fit

- [ ] Venue is welcoming to the chapter's riders and guests.
- [ ] Noise, parking, and neighborhood impact are considered.
- [ ] Venue expectations are clear to attendees.
- [ ] The chapter can leave the space better than it found it.

## Documentation

- [ ] Venue record is created from [Templates/Venue.md](Templates/Venue.md).
- [ ] Event record links to the venue when applicable.
- [ ] Post-event notes are added after the first visit.
- [ ] Rating is assigned only after leadership has enough direct experience.
