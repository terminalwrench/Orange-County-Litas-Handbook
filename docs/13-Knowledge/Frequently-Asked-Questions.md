# Frequently Asked Questions

Use this file for questions that come up repeatedly across riders, leaders, and event organizers. Keep answers short and link to the deeper module when the answer depends on policy, route details, or current event information.

## New Riders

### Do I need to know someone before attending?

No. Arriving alone is common. See [../01-Chapter/New-Rider-Guide.md](../01-Chapter/New-Rider-Guide.md) for first-event expectations.

### How do I know whether a ride is beginner friendly?

Check the event description for route difficulty, pace, freeway or canyon exposure, distance, regroup points, and whether a Ride Lead and Sweep Rider are identified. See [../02-Events/Beginner-Rides.md](../02-Events/Beginner-Rides.md) and [../05-Routes/Route-Planning-Standards.md](../05-Routes/Route-Planning-Standards.md).

### What if I am nervous about joining a ride?

Ask the host, Ride Lead, or Sweep Rider about the route before deciding. It is okay to meet at the destination instead of riding with the group.

### What should I bring?

Bring the items you need to ride legally, comfortably, and safely. For rides, arrive fueled unless the event specifically says otherwise. See [../01-Chapter/New-Rider-Guide.md](../01-Chapter/New-Rider-Guide.md).

### What if I am running late?

For social events, arrive when you can. For rides, do not chase the group. Meet at the destination if that information is available.

## Event Organizers

### What does an event need before it is announced?

At minimum: date, time, location, host, event type, ride status, expected rider level if riding, weather plan when relevant, and clear public language. See [../02-Events/Event-Lifecycle.md](../02-Events/Event-Lifecycle.md).

### How do I choose a venue?

Review [../02-Events/Venue-Selection.md](../02-Events/Venue-Selection.md), [../11-Venues/Venue-Index.md](../11-Venues/Venue-Index.md), and [../11-Venues/New-Venue-Evaluation.md](../11-Venues/New-Venue-Evaluation.md).

### How do I capture lessons after an event?

Use [Lessons-Learned.md](Lessons-Learned.md) for reusable lessons and update the event record in [../10-Events-Database](../10-Events-Database/README.md) with event-specific notes.
