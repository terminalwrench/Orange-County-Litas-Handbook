# Social Media Guidelines

Use this page before publishing an event post, story, reel, recap, or chapter announcement. Keep it practical: verify details, credit people correctly, and make the post easy for riders to act on.

## Sizes

| Format | Recommended Setup |
| --- | --- |
| Instagram Post | 1080 x 1350 px for portrait feed posts when possible. Use 1080 x 1080 px only when a square layout is intentional. |
| Instagram Story | 1080 x 1920 px. Keep key text away from the top and bottom interface areas. |
| Instagram Reel | 1080 x 1920 px. Keep the first frame readable, avoid tiny text, and make the event detail visible without sound. |

## Required Event Details

Include when applicable:

- Event name.
- Date and day of week.
- Start time.
- Meetup time and kickstands-up time for rides.
- Location or destination.
- Address, RSVP, or registration instructions.
- Ride difficulty or beginner-friendly status.
- Cost, donation, or ticket requirement.
- Parking note.
- Weather or cancellation note.
- Host, partner, venue, sponsor, or collaborator names.

## Publishing Checklist

- [ ] Date, time, location, and address are verified.
- [ ] Rider requirements are clear.
- [ ] Partner, sponsor, venue, and photographer tags are correct.
- [ ] The Litas and relevant nearby chapters are tagged when appropriate.
- [ ] Caption repeats important details that appear in the graphic.
- [ ] Text is readable on mobile.
- [ ] Alt text, captions, or accessible description are included when practical.
- [ ] Final proofread is complete.
