# Posting Workflow

The posting workflow keeps social content accurate, timely, and easy to archive.

## Workflow

1. Confirm the purpose of the post.
2. Confirm owner, deadline, and publishing channel.
3. Pull event details from the event brief when applicable.
4. Request or confirm final media through [../03-Media](../03-Media/README.md).
5. Check required tags, collaborator accounts, and credits.
6. Review accessibility needs.
7. Schedule or publish.
8. Verify the live post.
9. Archive the final asset and caption.

## Accuracy Check

Before posting, confirm:

- Date
- Time
- Location
- Ride status
- RSVP or payment details
- Weather or cancellation note
- Partner or sponsor approval
- Required tags

For event changes, follow [../02-Events/Cancellation-Policy.md](../02-Events/Cancellation-Policy.md).
