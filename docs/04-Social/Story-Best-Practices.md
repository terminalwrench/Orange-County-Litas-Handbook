# Story Best Practices

Stories are useful for reminders, quick updates, day-of coverage, and resharing attendee content.

## Operational Uses

Use stories for:

- Event reminders
- Parking or timing notes
- Weather updates
- Day-of arrivals
- Recap highlights
- Partner thanks
- Reposts from attendees

## Checks

Before posting:

- Confirm time-sensitive details.
- Keep important text away from interface controls.
- Tag accounts accurately.
- Avoid sharing unsafe behavior.
- Save final story assets when they are part of an event record.
