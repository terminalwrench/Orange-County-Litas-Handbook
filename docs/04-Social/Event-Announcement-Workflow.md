# Event Announcement Workflow

Use this workflow for social announcements tied to OC Litas events.

## Before Announcement

Confirm:

- Event owner
- Event type
- Date and time
- Location
- Ride status
- Beginner-friendly status
- RSVP or payment needs
- Weather plan
- Flyer or media asset
- Required tags

The event should be far enough through [../02-Events/Event-Lifecycle.md](../02-Events/Event-Lifecycle.md) that the public details are stable.

## Announcement Package

Save:

- Final flyer or asset
- Caption
- Required tags
- Story version if used
- Email or newsletter reference if used

## After Publishing

- Verify the post.
- Answer early questions.
- Watch for incorrect details.
- Archive the final asset and caption.
