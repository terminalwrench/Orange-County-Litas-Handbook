## Review Standard

Before using a hashtag:

- Confirm it is still active and appropriate.
- Confirm it does not attach the chapter to unrelated or harmful content.
- Avoid partner or campaign hashtags without context.
- Keep event-specific campaign tags documented with the event record.
- Use a short, relevant set instead of a long list no one checks.
