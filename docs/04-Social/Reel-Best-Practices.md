# Reel Best Practices

Reels should be organized as publishable media assets with clear ownership, final exports, captions, covers, and archive links.

## Before Editing

Confirm:

- Purpose of the reel
- Event or campaign
- Source clips
- Required tags
- Credit requirements
- Delivery date

## Before Publishing

Use [../03-Media/Templates/Reel-Checklist.md](../03-Media/Templates/Reel-Checklist.md). Confirm final export, cover, caption, tags, accessibility captions, and archive destination.

## After Publishing

Save the final video, cover, caption, posting date, tagged accounts, and any collaborator status.
