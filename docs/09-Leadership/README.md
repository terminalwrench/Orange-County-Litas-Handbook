This module is a focused leadership workspace for OC Litas. It supports light planning and meeting capture without turning the portal into project management software.

Use this section for leadership roles, meeting notes, yearly planning, and working ideas that are not ready for broader use.

## Privacy Standard

This section is intended for leadership use. Do not store sensitive personal details, private conflicts, payment records, medical information, or incident details in broad handbook files. When private context is needed, summarize the operational takeaway and keep sensitive details in the appropriate restricted channel.
