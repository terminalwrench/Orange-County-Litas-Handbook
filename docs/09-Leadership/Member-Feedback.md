# Member Feedback

Use this file to track feedback themes from members while protecting privacy.

## Feedback Principles

- Capture themes, not gossip.
- Separate operational learnings from personal details.
- Do not store sensitive private information.
- Look for repeated patterns across events.
- Route urgent safety or conflict issues through leadership immediately.

## Feedback Theme Format

```markdown
## YYYY-MM-DD - Theme

- Source:
- Summary:
- Related event or module:
- Suggested action:
- Owner:
- Status:
```

## Common Feedback Areas

- Event clarity
- Ride pace
- Beginner support
- Venue fit
- Parking
- Social communication
- Welcoming experience
- Photo or media delivery

When feedback changes a repeatable process, update the related handbook module.
