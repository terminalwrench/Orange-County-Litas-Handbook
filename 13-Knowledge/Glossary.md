# Glossary

This glossary defines chapter terminology, acronyms, and common phrases used across the handbook.

## Terms

| Term | Meaning |
| --- | --- |
| Chapter | The local Orange County Litas group. |
| OC Litas | Orange County Litas. |
| Meet & Greet | A social gathering focused on connection rather than a full ride. |
| M&G | Short form for Meet & Greet. |
| Ride Lead | The rider responsible for leading the group ride and route decisions. |
| Sweep Rider | The rider assigned to stay at the back of the group and help keep the ride together. |
| Kickstands Up | The planned departure time for a ride. |
| Regroup | A planned stop or pause where the group reconnects before continuing. |
| Destination-only option | An option for someone to meet at the destination instead of joining the group ride. |
| Beginner-friendly | A ride or event designed to support newer riders; not a guarantee of no risk. |
| Intermediate ride | A ride requiring more comfort with distance, traffic, freeway, curves, or group riding. |
| Pre-ride | To test or review a route before using it for a chapter event. |
| Event owner | The person responsible for coordinating an event from planning through follow-up. |
| Collaboration | An event or effort coordinated with another chapter, partner, sponsor, venue, creator, or community group. |
| Community event | An outside motorcycle event OC may attend, support, or track without hosting. |
| Recap | Post-event summary, often used for documentation, social content, or lessons learned. |
| Run of show | A simple timeline or sequence for an event. |
| Canonical record | The primary source of truth for a piece of information. |
| Institutional knowledge | Useful chapter knowledge that should not depend on one person's memory. |
