# Favorite Stops

Use this file to track reusable stops that support rides and social events.

## Stop Criteria

Useful stops have:

- Safe entry and exit
- Legal parking
- Restrooms when possible
- Water or food access when possible
- Enough space for the group size
- A reason to return

## Stop Record

For each stop, record:

- Name
- Address
- Best use
- Parking notes
- Restroom notes
- Group size fit
- Photo considerations
- Issues to avoid
