# Route Checklist

Use this checklist before approving a route for an OC Litas ride.

- [ ] Route purpose is clear.
- [ ] Rider level is defined.
- [ ] Route has been tested or locally verified.
- [ ] Road surface is acceptable.
- [ ] Construction has been checked.
- [ ] Traffic timing has been considered.
- [ ] Gas plan is known.
- [ ] Restroom access is known.
- [ ] Regroup points are identified.
- [ ] Weather exposure has been reviewed.
- [ ] Destination parking is confirmed.
- [ ] Lead and sweep can explain the route.
- [ ] Alternate route or destination-only plan is available when needed.
