# Scenic Routes

Scenic routes are useful for chapter rides, destination events, and photo-friendly outings. Scenic value should never override safety, rider fit, or timing.

## Evaluation

For each scenic route, document:

- Road type
- Distance
- Turn complexity
- Traffic patterns
- Pullouts
- Photo stops
- Gas and restrooms
- Weather exposure
- Beginner suitability

## Use

Pair scenic routes with [Route-Checklist.md](Route-Checklist.md) before adding them to an event plan.
