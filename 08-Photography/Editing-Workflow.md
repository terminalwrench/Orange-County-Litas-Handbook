# Editing Workflow

The editing workflow keeps raw files, selects, edits, and final deliveries organized.

## Workflow

1. Import files by event date and name.
2. Separate raw files from selects.
3. Choose selects for editing.
4. Export finals for intended use.
5. Name final files consistently.
6. Save credit and usage notes.
7. Archive final delivery links.

Use [../03-Media/Naming-Conventions.md](../03-Media/Naming-Conventions.md) for file names.
