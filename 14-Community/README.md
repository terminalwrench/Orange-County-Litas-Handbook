# Community Events

This module tracks motorcycle events, Litas chapter activities, partner gatherings, industry weekends, charity rides, and training opportunities that Orange County Litas attends, supports, monitors, or collaborates with but does not necessarily host.

This is not a replacement for the canonical [Events Database](../12-Events-Database/README.md). Event-specific OC Litas records belong in the Events Database. This module explains the wider community context: why an event matters, when it usually happens, who hosts it, how OC has participated, and what leadership should consider when planning around it.

## Source Standard

Initial records are based on the existing Events Database and the original calendar export history. Do not invent attendance, partnerships, venues, or commitments. If an event was only present in calendar history, record that as calendar history rather than chapter participation.

## Maintenance

Update this module when:

- OC Litas attends, supports, or plans a ride to a community event.
- A recurring event changes month, location, host, or format.
- A partner relationship becomes stronger, weaker, or inactive.
- A calendar conflict affects OC event planning.
- Leadership learns participation notes that future planners should know.

Use [../13-Venues](../13-Venues/) for location-specific notes and [../05-Routes](../05-Routes/) for route knowledge when OC plans a ride to a community event.
